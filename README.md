В данном проекте было реализовано построение точных доверительных интервалов для параметров нормального распределения $\mathcal{N}(a, \sigma^2)$ в четырёх вариантах:
1. ДИ для $a$ при известном $\sigma$
2. ДИ для $\sigma^2$ при известном $a$
3. ДИ для $a$ при неизвестном $\sigma$
2. ДИ для $\sigma^2$ при неизвестном $a$

Построение ДИ основано на так называемом методе центральной статистики.

Пусть у нас есть параметрическое семейство распределений
$(\mathrm{P}_{\theta})_{\theta}$ и мы хотим оценить параметр $\theta$.

Обозначим выборку через $X = (X_1, \ldots, X_n)$.

Если у нас получилось найти такую функцию $G(X, \theta)$, что её распределение не зависит от $\theta$, мы можем взять $g_1$ и $g_2$ - квантили распределения $G(X, \theta)$ уровня $\gamma_1$ и $\gamma_2$ соответственно.

В такой ситуации $\forall \theta \in \Theta$ будет выполнено:\
 $\mathrm{P}_{\theta}(g_1 \leqslant G(X, \theta) \leqslant g_2) ) \geqslant \gamma_2 - \gamma_1$.

Если мы сможем разрешить неравенство $g_1 \leqslant G(X, \theta) \leqslant g_2$ относительно $\theta$ и выберем $\gamma_1$ и $\gamma_2$ такими, что $\gamma_2 - \gamma_1 = \epsilon$, то мы получим доверительную область для $\theta$ уровня доверия $\epsilon$.
